외국인을 위한 국내 여행 정보 사이트

KH 정보 교육원에서 진행한 SEMI PROJECT

프로 젝트 : 외국인을 위한 국내 여행 정보 사이트
개발 기간 : 2019.05.27 ~ 2019.06.21 (총 5명)
개발 목표 : 외국인들이 우리나라를 여행을 하게 되었을 때 우리나 전통미를 강조하며
                      다른 사이트와 다른 편리함과 다양한 정보를 제공
                      우리나라의 축제 및 관광지 등의 정보를 쉽게 수집 할 수 있도록 할 것 이다.
개발 환경
-         OS      : Window 7 , 10 / Mac OS
- 구현 언어 : Java, HTML5, CSS3, Java Script, Oracle
- 사용 기술 : Ajax, Servlet, JSP, Google Map API, Facebook Login API, Kakao (Pay, Map) API
                         공공데이터포털 데이터 API, 기상청(날씨누리) API
- 사용 도구 : Eclipse, Oracle DataBase 11g, Apache Tomcat 9.0 , Summer Note
구현 기능 : 로그인, 결제, 지도, 게시판, 채팅, 날씨, 투어, 축제, 관광지
