너만의 도시락 나만의 도시락

KH 정보 교육원에서 진행한 FINAL PROJECT

프로 젝트 : 커스텀 도시락 제작 판매 사이트
개발 기간 : 2019.08.02 ~ 2019.09.02 (총 6명)
개발 목표 : 바쁜 현대 사회에 직장인들이 식사를 잘 챙겨 먹지 못하고 있다.
                      이러한 이유로 커스텀 도시락 제작 판매 홈페이지를 제작하여 정해진 시간에 원하는 반찬으로
                      이루어진 도시락을 지정된 장소에서 받아 규칙적인 식사를 돕는다.
개발 환경
-         OS      : Window 7 , 10 / Mac OS
- 구현 언어 : Java, HTML5, CSS3, Java Script, Oracle
- 사용 기술 : Ajax, 주소 API, 캡차, QR코드, 그래프 API(D3.js), WebNotification API, Mail SMPT
                         kakao (Login, Pay, Map) API, Spring (Batch, Security, WebSocket), jQuery(Drag and Drop)
- 사용 도구 : Eclipse, Oracle DataBase 11g, Apache Tomcat 9.0 , Bootstrap4.0 , Summer Note
구현 기능 : 로그인, 결제, 지도, 검색, 채팅, 커스텀, QR코드, 장바구니
